# 跨学科知识库

## 简介
这是一个整合行为经济学、数据科学和运筹学的个人知识库，主要基于以下学习经历：
- 华威大学行为经济学硕士
- 华威大学MORSE本科
- UMASS DACSS硕士
- 相关领域公开课程
